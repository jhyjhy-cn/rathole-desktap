# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.0.1] - 2026-01-20

### Added
- Initial release of Rathole Desktop.
- Core architecture using Tauri 2 and Vue 3.
- Visual configuration editor for Proxy services.
- Process management (Start/Stop) with log streaming.
- Built-in Download Manager for rathole binaries.
- System Tray integration.
- Dark mode UI using Element Plus.
