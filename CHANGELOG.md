# 更新日志

本项目的所有重要更改都将记录在此文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
并且本项目遵循 [Semantic Versioning](https://semver.org/lang/zh-CN/spec/v2.0.0.html)。

## [0.0.1] - 2026-01-20

### 新增
- Rathole Desktop 初始版本发布。
- 基于 Tauri 2 和 Vue 3 的核心架构。
- 代理服务的可视化配置编辑器。
- 进程管理（启动/停止）及日志流。
- 内置 Rathole 二进制文件下载管理器。
- 系统托盘集成。
- 使用 Element Plus 实现的暗色/亮色模式 UI。
- 国际化支持（简体中文/英文）。
